# Job 1: Forward Deployed Engineer (AI-Native) — Newpage (PLACEHOLDER)

## About the role
Newpage is looking for a Forward Deployed Engineer to work directly with customers,
building AI-native applications end to end. You will own solutions from discovery
through deployment, working hands-on with LLMs, retrieval pipelines, and modern
full-stack tooling.

## Responsibilities
- Build and deploy full-stack AI applications for enterprise customers.
- Design RAG pipelines: ingestion, chunking, embedding, retrieval, grounding.
- Integrate LLM providers with attention to cost, latency, and quality tradeoffs.
- Work directly with stakeholders to scope pragmatic solutions under time constraints.
- Containerise and ship production services.

## Requirements
- Strong TypeScript/JavaScript and a modern web framework (Next.js, React).
- Experience with PostgreSQL and vector search.
- Hands-on experience building with LLM APIs (OpenAI, Anthropic).
- Docker and cloud deployment experience (AWS, GCP, or Azure).
- Excellent written communication; ability to explain technical decisions.
